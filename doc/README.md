## Features Overview

### Batch Processing
- **Compressor**: Upload multiple files, run batch compression
- **Resizer**: Folder watching with automatic batch processing
- **Font Converter**: Convert multiple fonts at once
- **PDF Bulk Downloader**: Range-based batch PDF generation with concurrent processing

### Manual Processing
- **Comparer**: Upload any two PDFs to compare
- **Image Duplicator**: Upload CSV + images for duplication
- **Resizer**: Direct upload with dimension selection
- **Thumbnailer**: Upload PDFs for thumbnail generation

### Output Formats
- **Compression**: Original format maintained with quality optimization
- **Font Conversion**: TTF, WOFF, WOFF2, SVG, EOT in a single ZIP
- **Image Duplication**: All copies with original extension preserved
- **Image Resizing**: JPEGs in center/top/bottom crop variants
- **Thumbnails**: High-resolution JPEGs (3:4 aspect ratio)
